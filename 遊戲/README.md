# CyberPose 雙人體感姿勢挑戰賽

這是一款以 HTML5、CSS3 與純 JavaScript 開發的網頁雙人體感對戰遊戲。
靈感來自於日本經典綜藝節目「超級變變變 / 牆壁來了 (Hole in the Wall)」。玩家只需透過電腦的網路攝影機，不需安裝任何額外應用程式，即可與朋友進行即時的姿勢 PK 挑戰！

## ✨ 遊戲特色

* **純前端運作**：無需後端伺服器，直接利用本機資源執行，保障隱私且零延遲。
* **AI 骨骼追蹤技術**：整合 Google 的 TensorFlow.js 與 MoveNet Multipose Lightning 模型，能夠精準且快速地同時捕捉畫面上多位玩家的 17 個關鍵骨骼點。
* **獨立分割畫面 (Split-Screen)**：畫面正中央配有發光分隔線，左右玩家各擁有一半的遊戲區域，並會面對各自隨機且不同的圖框挑戰，避免互相干擾。
* **20+ 種豐富關卡**：內建包含「暴龍」、「展翅雄鷹」、「狹縫求生」等 20 種漸進式難度關卡，系統配有「防重複」洗牌機制，確保一場 10 回合的對戰中，絕對不會出現重複的圖框！
* **像素級精準判定 (Pixel-Perfect Collision)**：不拘泥於傳統的幾何方形判定，即使是極為複雜的不規則圖形，只要玩家的肢體落在發光線條範圍內，系統都能精準給分。
* **小畫家自創圖庫**：內建「繪圖工坊」，玩家可以自由使用滑鼠拖曳畫出任何奇形怪狀的破洞，存檔後將自動成為下一次對戰中的隨機關卡。

## 🚀 如何開始遊戲

由於現代瀏覽器基於安全性考量 (CORS)，不允許直接點擊 `index.html` (即 `file:///` 協定) 來下載 AI 權重模型，因此請透過本地伺服器來執行遊戲：

### 方法一：使用內建啟動器 (Windows 推薦)
直接雙擊專案資料夾內的 **`開始遊戲.bat`**。
它會自動偵測您的環境（Python 或 Node.js），架設好本地伺服器，並自動在瀏覽器中開啟遊戲！

### 方法二：使用 VS Code
1. 開啟 Visual Studio Code。
2. 將此專案資料夾拖曳進 VS Code。
3. 安裝擴充功能 **Live Server**。
4. 點擊畫面右下角的 **Go Live** 按鈕即可開始遊玩。

### 方法三：使用終端機指令
如果您熟悉命令列，可以在此資料夾目錄下執行：
* Node.js: `npx http-server -p 8080 -c-1`
* Python 3: `python -m http.server 8080`

然後在瀏覽器網址列輸入 `http://localhost:8080` 即可。

## 🕹️ 遊玩說明

1. 進入遊戲後，請同意瀏覽器存取「攝影機」的權限。
2. 等待 AI 模型載入完成（約需 3~5 秒）。
3. 兩位玩家請分別站立在畫面的左半邊與右半邊。
4. 點擊 **START GAME**，畫面的左右兩邊會開始飛出各種奇形怪狀的發光框框。
5. 盡可能將身體（包含頭部與四肢）凹進您的安全框框內！越吻合框框的形狀，獲得的分數越高。
6. 10 回合結束後，總分最高的玩家獲勝！

## 🛠️ 技術堆疊 (Tech Stack)

* **HTML5 / CSS3**：建構 Cyberpunk / 霓虹街機風格的使用者介面。
* **Vanilla JavaScript (ES6)**：處理核心遊戲邏輯、像素級判定演算與自創圖形引擎。
* **Canvas API**：繪製流暢的 3D 視覺縮放特效與處理碰撞遮罩。
* **TensorFlow.js & MoveNet**：提供穩定、高幀率的機器學習人體姿態辨識 (Human Pose Estimation)。
